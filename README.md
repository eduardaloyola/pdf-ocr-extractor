# pdf-ocr-extractor

# 📄 Nota Fiscal OCR Automation

Projeto desenvolvido em Python para automatizar a extração de informações de notas fiscais em PDF digitalizadas.

## Funcionalidades

- Conversão de PDF para imagem
- Reconhecimento óptico de caracteres (OCR)
- Extração automática de:
  - Número da nota
  - Valor total
  - Entidade
  - Unidade
- Exportação dos resultados para arquivo `.txt`

## Tecnologias utilizadas

- Python
- EasyOCR
- PyMuPDF
- NumPy
- Regex
